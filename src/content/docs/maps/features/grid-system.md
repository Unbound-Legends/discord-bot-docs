---
title: Grid System
description: Configure square or hex grids for tactical gameplay and tiled maps
---

:::note[Stub]
This feature page will cover:
- Grid types available:
  - Square grid
  - Hexagon (pointy-top)
  - Hexagon rotated (flat-top)
- Configuring grid visibility and color
- Setting grid cell size (50-500 pixels)
- Snap-to-grid: hold Shift while dragging to snap asset centers to grid cells
- Arrow key movement: move selected assets one grid cell at a time
- Hex coordinate math: how hex grids handle diagonal movement
- Using grids for tiled maps:
  - Aligning map tile images to the grid
  - Setting cell size to match your tile dimensions
  - Auto-resize to grid option for new assets
- Per-page grid settings (each map page can have different grid config)
:::
