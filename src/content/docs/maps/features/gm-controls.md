description: Settings and tools available only to Game Masters
---

:::note[Stub]
This feature page will cover:
- The Settings panel (gear icon): General tab vs GM-only tabs
- NPC shadow behavior modes (how concealed NPCs appear to players):
  - Show normally: full opacity, name visible
  - Hide name only: full opacity, name hidden
  - Hide name + lower opacity: 30% opacity, name hidden
  - Hide completely: invisible to players
- Layer management:
  - Creating new layers
  - Reordering layers (affects what renders on top)
  - GM-only layers (visible only to GM)
  - Moving assets between layers
- Asset visibility controls:
  - Hidden: GM sees at 50% opacity, players don't see at all
  - Restricted: GM-only visibility flag
  - Locked: prevents accidental movement
- Clearing exploration (resets what players have seen)
- GM sees through fog at reduced opacity
- Real-time broadcasting: changes sync instantly to all players
:::
