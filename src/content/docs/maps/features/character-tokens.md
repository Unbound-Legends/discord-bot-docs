title: Character Tokens
description: Link tokens to characters to display stats, track wounds, and manage visibility
---

:::note[Stub]
This feature page will cover:
- Linking tokens to RPG Sessions characters
- The stats display: wounds bar (red) and strain bar (blue)
- How wound/strain thresholds affect the bars
- Minion groups: the elimination counter showing "3/5" style display
- How minion elimination is calculated (wounds exceeding threshold)
- Visibility levels:
  - Complete: name + stats visible to players
  - Known: name visible, stats hidden
  - Visible: token visible, name/stats hidden
  - GM-only: only the GM can see the token
- Character name labels (floating below tokens)
- Flipping tokens: Shift+F (horizontal), Shift+V (vertical)
:::
