:::note[Stub]
This reference page will list all global keyboard shortcuts:

**Tool Selection**
- V - Pointer tool (select and move assets)
- Z - Move camera tool (pan the view)
- B - Draw tool (freehand drawing)
- E - Eraser tool (erase drawings)
- H - Lasso tool (multi-select assets)

**Asset Tools** (when asset is selected)
- R - Rotate mode (drag to rotate)
- L - Scale mode (drag to resize)

**GM-Only Tools**
- F - Fog of war tool

**Clipboard**
- Ctrl/Cmd + C - Copy selected assets
- Ctrl/Cmd + V - Paste assets

**Undo**
- Ctrl/Cmd + Z - Undo last action

**Grid Movement** (when asset is selected and grid is enabled)
- Arrow keys - Move asset one grid cell in that direction
- Shift + Arrow keys - Move asset five grid cells in that direction

**Asset Manipulation** (when asset is selected)
- Shift + F - Flip horizontally
- Shift + V - Flip vertically
- Shift + drag - Snap to grid

**Selection**
- Shift + click - Add/remove from selection
- Ctrl/Cmd + click - Toggle individual asset in selection
:::
